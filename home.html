It contains three main Guess Game
Play With Number
This Game has three input box.
Box1 will accept Lower Range
Box2 will accept Upper Range
Box3 will accept the number guessed by you in between the range of [Box1,Box].
if your guess is correct.it will alert good guess Unless it will alert wrong answer and then the right answer too.


Play With Capital Letter
This Game has three input box.
Box1 will any Capital Letter Alphabet.
System will generate an Alphabet.
if your guess is correct.it will alert good guess
Unless it will alert wrong answer and then the right answer too.

Play With Small Letter
This Game has three input box.
Box1 will any Small Letter Alphabet.
System will generate an Alphabet.
if your guess is correct.it will alert good guess
Unless it will alert wrong answer and then the right answer too.
